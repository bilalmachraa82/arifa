## 5. Implementacao

### Meta tags por pagina
Cada pagina principal tem SEO.tsx configurado com:
- `title`: keyword primaria + brand
- `description`: keyword primaria + secundaria + CTA
- `keywords`: lista de keywords relevantes

### Conteudo optimizado
Os 3 artigos piloto em /insights/ foram criados para targetar clusters de keywords:
1. **guia-construcao-portugal** → cluster Particulares
2. **reabilitacao-eficiencia-energetica** → cluster Empresas
3. **investimento-imobiliario-lisboa** → cluster Investidores

### Schema markup
Cada artigo inclui:
- Article schema (JSON-LD)
- FAQ schema com perguntas reais do segmento
- BreadcrumbList schema
- Organization schema (global)

### Proximos passos
1. Validar volumes reais com Google Keyword Planner ou Ahrefs
2. Monitorizar rankings iniciais (Google Search Console)
3. Criar mais conteudo para long-tail keywords de alto valor
4. Optimizar conteudo existente com base em dados de Search Console
